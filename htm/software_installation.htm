Software Installation

To install the Rotating Manipulator software, follow these instructions:

  1. Use the flash drive included with the Rotating Manipulator, or download the software from VideoRay's software download page.
  2. Double click rotating_manipulator.exe. This will copy the files to your local hard drive.

Prior to using the Rotating Manipulator, you need to configure the software to specify whether you are using the Rotating Manipulator, Cutter or standard manipulator.
